# Server Parsing

Express.js сервер для парсинга торрент-файлов с веб-сайтов.

## Описание

Этот проект представляет собой сервер на Express.js, который позволяет парсить торрент-файлы с указанных веб-сайтов. Сервер предоставляет API endpoints для получения и обработки торрент-данных.

## Установка

1. Клонируйте репозиторий или скачайте проект
2. Установите зависимости:

```bash
npm install
```

## Настройка

1. Скопируйте файл `.env` и настройте переменные окружения при необходимости
2. По умолчанию сервер запускается на порту 3000

## Запуск

### Режим разработки (с автоматической перезагрузкой)

Для разработки используйте команду `dev`, которая автоматически перезапускает сервер при изменении файлов:

```bash
npm run dev
```

Сервер будет автоматически перезапускаться при любых изменениях в коде. Это удобно во время разработки.

### Продакшн режим

Для запуска в продакшн режиме используйте:

```bash
npm start
```

Сервер будет доступен по адресу `http://localhost:3000`

## Работа с Postman

### Установка Postman

1. Скачайте Postman с официального сайта: https://www.postman.com/downloads/
2. Установите приложение (доступны версии для Windows, Mac и Linux)
3. Создайте бесплатный аккаунт или войдите в существующий

### Первые шаги

1. **Запустите сервер в режиме разработки**:
   ```bash
   npm run dev
   ```
   Сервер будет доступен по адресу `http://localhost:3000` и автоматически перезапустится при изменениях

2. **Откройте Postman** и создайте новый запрос:
   - Нажмите кнопку "New" → "HTTP Request"
   - Или используйте сочетание клавиш `Ctrl+N` (Windows) / `Cmd+N` (Mac)

3. **Настройте запрос**:
   - **Метод**: Выберите `GET` из выпадающего списка
   - **URL**: Введите `http://localhost:3000/`
   - Нажмите кнопку **Send**

4. **Проверьте ответ**:
   - В нижней части окна вы увидите ответ сервера
   - Статус код должен быть `200 OK`
   - В теле ответа будет текст: `Hello World!`

### Полезные функции Postman

- **Сохранение запросов**: Нажмите "Save" чтобы сохранить запрос в коллекцию
- **Создание коллекций**: Группируйте связанные запросы в коллекции для удобства
- **Переменные окружения**: Используйте переменные для базового URL (например, `{{baseUrl}}/`)
- **История запросов**: Все отправленные запросы сохраняются в истории

### Примеры запросов

#### GET запрос к корневому эндпоинту
```
GET http://localhost:3000/
```

**Ожидаемый ответ:**
- Status: `200 OK`
- Body: `Hello World!`
